C++ source code documentation¶

Explanations of the C++ source.

macrodef.h¶

All macro definitions are located in the macrodef.h file. With these macro definitions we control the functionality of the whole system. Definitions should be set here before compiling the parfis library.

global.h¶

std::string parfis::tostr(int num)¶

Functions tostr() convert from a type (given in argument) to string.

std::vector<std::string> parfis::stovecs(const std::string &str, char bra = '{', char ket = '}', char sep = ',')¶

Converts string to a vector of strings.

Function returs vector of strings that is parsed between bra and ket characters, with sep as a separator. Default values for bra and ket are ‘{‘ and ‘}’ characters and the separator is ‘,’. For example stovecs(“{a,b,c}”) returns vector {“a”, “b”, “c”}.

std::string parfis::cleanString(const std::string &str)¶

Removes spaces, tabs and commend lines

Comment line starts with ‘#’ and ends with newline.

std::string parfis::getStringValue(const std::string &data, const char *key)¶

Function for parsing ‘key=value’ strings.

If the key doesn’t exist or not properly formated, returns empty string.

std::string parfis::getStringSection(const std::string &data, const char *key)¶

Function for parsing ‘[section]’ strings.

If the key doesn’t exist or not properly formated, returns empty string. Section strings are used in the initialization files.

template<class T>
T parfis::toValue(const std::string &data, const char *key)¶

Template that converts data string to a value

Implementation of the template specializes for certain types T. The data string is of the format ‘key=value’, and the key is the the name of the ‘key’. Internaly this function calls getStringValue().

template<class T>
struct parfis::XYZ¶

Struct that represents a 3d vector with components of type T.

The struct has three member variables x, y and z of type T.

Public Functions

inline T len()¶

Returns the vector length in Eucledian metric.

The length is calculated as \( \sqrt{x^2 + y^2 + z^2} \)

inline T lenSquared()¶

Returns the squared vector length in Eucledian metric.

The length is calculated as \( x^2 + y^2 + z^2 \)

parfis.h¶

struct ParfisAPI¶

Parfis interface class.

Interface class that is exported in order to be accsessible from outside this library. Since this is a C++ struct, it can be only accessed from another C++ code.

Subclassed by parfis::Parfis

group extern_c

C Functions implementation of the interface class ParfisAPI.

Functions are compiled with the extern “C” thus forbiding C++ to mangle the function names. This is essential for using these functions with python’s ctypes.

Data structures and types¶

Data structures and types that are used all over the system are declared and defined in the file datastruct.h/datastruct.cpp.

type state_t¶

Type of the particle state variables position and velocity. This can be set to double or float by setting the STATE_NUMERIC_TYPE.

Note

When set to double the working application occupies approximately two times more RAM, than when float is set.
