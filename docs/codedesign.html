Software design and organization¶

The main concept of the parfis code is to have all the functionality inside the shared library which is interfaced with the use of python code (using ctypes). Googletests are run by linking to the shared library and invoking functions that are also accessible from python’s ctypes.

Design of the parfis library¶

Parfis library is written in C++. Access to the library functions and classes is through the exported class parfis::ParfisAPI. The API can be used from C++ applications. In order to forbid C++ to mangle the function names we compile a set of functions with the extern “C” directive. C functions are exported and named in the format ParfisAPI_functionName. Functions’ main use is for calling them from ctypes.
