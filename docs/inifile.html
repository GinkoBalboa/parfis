# Parfis configuration file

[system]
geometry = cylindrical
timestep = 1.0e-3
E_field_type = none
B_field_type = none
    
[space]
geometry_size = (0.02, 0.02, 0.4)
unit_cell_size = (2e-3, 2e-3, 2e-3)
periodic_boundary = (0, 0, 0)

[particle]
name = {a}
particles_per_cell = {100}
timestep_ratio = {1}

[system]¶

Parameters in this section affect the whole system.

geometry: string, values: [cylindrical, cubical]

The geometry of the container.

timestep: double, unit: s

The smallest timestep in the system, all particle timesteps are smaller or equal to this value. Particle timesteps are given as a integers, which is a multiple of this value.

[space]¶

Parameters affect the space configuration, mainly concerned with the SpaceAPI class.

geometry_size: double xyz tuple, unit: m

Size of the simulation space in meters (z is the longitudal axis). If the geometry is defined as cylindrical, then x and y components must be equal.

unit_cell_size: double xyz tuple, unit: m

Size of a unit cell in meters. All other cell sizes are multiples of this size.

periodic_boundary: int xyz tuple, unit: boolean

Periodic boundary condition. The coresponding axis top and bottom boundaries are connected. This means that particles leaving the top boundary emerge from the bottom boundary. Value is boolean.

[particle]¶

Parameters affect particle specie configuration, mainly concerned with the ParticleAPI class.

name: string list

Specie names, can be any string.

particles_per_cell: uint list

Particles per unit cell initially.

timestep_ratio: uint list

Number of system.timestep that represents one timestep for the given specie.
